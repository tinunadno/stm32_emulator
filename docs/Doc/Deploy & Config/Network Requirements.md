# Сетевые требования для GDB

Для работы удаленной отладки необходимо обеспечить сетевую доступность воркеров. 

## Требования

- Прямая маршрутизация от GDB-клиента до IP воркера. 
- Открытые TCP-порты в диапазоне, выделенном под GDB (например, 1234–1244). 
- Firewall-правила, которые разрешают доступ только доверенным IP. 
- Опционально, VPN или SSH-туннель для дополнительной безопасности. 

Схема:

```text
Developer (GDB) ---- TCP ---- Worker (GDB server)
  IP: 203.0.113.42          Port: 12345
```
