# STM32 Online Lab — Обзор

Эта документация описывает систему онлайн-лаборатории для симуляции прошивок STM32 с поддержкой удаленной GDB-отладки. 

Система позволяет:
- загружать бинарные прошивки и запускать их на виртуальных платах;
- получать логи и телеметрию периферии (GPIO, UART, I2C, SPI) в реальном времени;
- подключаться GDB-клиентом к удаленному симулятору для пошаговой отладки. 

## Ключевые возможности

- **Асинхронный** пайплайн обработки задач поверх KeyDB (очереди + Pub/Sub). 
- **Горизонтальное масштабирование** за счет независимых воркеров. 
- **Отказоустойчивость**: при падении воркера задание возвращается в очередь. 
- **SSE-потоки** для браузерных клиентов. 
- **Remote GDB-отладка** через встроенный GDB-сервер в симуляторе. 
- **Наблюдаемость** через метрики и OpenTelemetry-трейсы. 

## Основные решения

- Хэширование бинарников по SHA-256 (для аудита и дедупликации). 
- SSE (`GET /events`) как протокол стриминга для клиентов. 
- Gateway отвечает только за планирование и маршрутизацию, а не выполнение. 
- Взаимодействие компонентов — через KeyDB, без прямых HTTP-вызовов. 
- Воркеры сами забирают задачи (pull-модель) и могут свободно масштабироваться.
